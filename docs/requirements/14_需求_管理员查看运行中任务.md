# 需求：管理员查看所有用户运行中任务

## 背景

当前管理员无法看到系统中有哪些用户在跑生成任务。需要加一个管理员面板，显示所有用户当前正在运行的任务状态。

## 详细规则

### 后端
- 在 `tasks.py` 新增 `GET /api/tasks/running` 接口（仅管理员可访问）
- 查询所有 `status == RUNNING` 的任务，关联 User 表获取用户名
- 返回字段：task_id, username, stage, model, progress_current, progress_total, created_at

### 前端
- 在配置中心"系统设置"标签页中新增"运行中任务"面板
- 以表格展示：用户名、阶段、模型、进度（current/total）、创建时间
- 仅管理员可见
- 自动刷新（每 5 秒轮询一次）

## 文件变更
| 文件 | 操作 | 说明 |
|------|------|------|
| `backend/app/routers/tasks.py` | 修改 | 新增 running 接口 |

| `frontend/src/views/ConfigCenter.vue` | 修改 | 新增运行中任务面板 |
